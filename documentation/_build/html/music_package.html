music_package.lyrics module¶

music_package.lyrics.get_lyric(artist, title, verbosity=True)¶

The function returns the lyrics of a song given the title and the artist.

Parameters

  • artist (string) – artist name

  • title (string) – title of the song

  • verbosity (bool) – verbosity activated

  • SONG_URL (string) – generic URL that will be implemented with input artist and title

Returns

lyrics of the chosen song

Return type

string
